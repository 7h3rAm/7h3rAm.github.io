SoriTong MP3 Player .m3u File Parsing SEH Overflow

¬´ üìÖ published on 02/Sep/2012 ¬ª

SoriTong MP3 Player version 1.0 is prone to a SEH overflow vulnerability. Processing specially-crafted .m3u file could trigger a SEH overwrite that could be leveraged further to gain arbitrary code execution. The exploit for this vulnerability has been documented at EDB: 8624

Here is a complete rewrite of this exploit:
